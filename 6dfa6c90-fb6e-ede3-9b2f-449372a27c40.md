
# SlicerCache.List Property (Excel)

 **True** if the slicer cache is for a slicer on a table. **False** otherwise. **Boolean** Read-only


## Version information

Version Added: Excel 2013 


## Syntax

 _expression_. **List**

 _expression_A variable that represents a  **SlicerCache** object.


## Property value

 **BOOL**
