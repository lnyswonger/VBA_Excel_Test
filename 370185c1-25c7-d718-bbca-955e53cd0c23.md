
# CalculatedMember.MeasureGroup Property (Excel)

Returns the associated measure group.  **String** Read-only


## Version information

Version Added: Excel 2013 


## Syntax

 _expression_. **MeasureGroup**

 _expression_A variable that represents a  [CalculatedMember](07a1f8df-107e-a5fd-3d15-dfc92916c4c6.md) object.


## Property value

 **STRING**
