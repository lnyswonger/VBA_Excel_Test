
# XlDataLabelSeparator Enumeration (Excel)

Specifies the separator used with data labels.



|**Name**|**Value**|**Description**|
|:-----|:-----|:-----|
| **xlDataLabelSeparatorDefault**|1|Excel selects the separator.|
