
# SparklineGroups.ClearGroups Method (Excel)

Clears the selected sparkline groups.


## Version Information

Version Added: Excel 2010 


## Syntax

 _expression_. **ClearGroups**

 _expression_A variable that represents a  ** [SparklineGroups](9bc6be34-fa2e-8652-ca92-fa9630b4d7a6.md)** object.


### Return Value

Nothing
