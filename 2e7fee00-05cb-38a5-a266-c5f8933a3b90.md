
# WorksheetFunction.Acot Method (Excel)

Returns the arccotangent of a number, in radians in the range 0 to Pi.


## Version information

Version Added: Excel 2013 


## Syntax

 _expression_. **Acot**(Arg1)

 _expression_A variable that represents a  [WorksheetFunction Object (Excel)](7b1d5639-363d-632c-2cf0-2232562646b6.md) object.


### Parameters



|**Name**|**Required/Optional**|**Data type**|**Description**|
|:-----|:-----|:-----|:-----|
|Arg1|Required|DOUBLE|The cotangent of the angle you want.|

### Return value

 **DOUBLE**
