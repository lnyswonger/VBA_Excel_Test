
# XlDataSeriesDate Enumeration (Excel)

Specifies the type of date to apply to a data series.



|**Name**|**Value**|**Description**|
|:-----|:-----|:-----|
| **xlDay**|1|Day|
| **xlMonth**|3|Month|
| **xlWeekday**|2|Weekdays|
| **xlYear**|4|Year|
