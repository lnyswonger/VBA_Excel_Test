
# DataLabel.Width Property (Excel)

Returns the width, in points, of the object. Read-only.


## Version Information

Version Added: Excel 2010 


## Syntax

 _expression_. **Width**

 _expression_A variable that represents a  **DataLabel** object.


### Return Value

Double
