
# XlTimeUnit Enumeration (Excel)

Specifies the unit of time for chart axes and data series.


## Version Information

Version Added: Excel 2007 



|**Name**|**Value**|**Description**|
|:-----|:-----|:-----|
| **xlDays**|0|Days|
| **xlMonths**|1|Months|
| **xlYears**|2|Years|
