
# ShapeRange.SoftEdge Property (Excel)

Returns a  ** [SoftEdgeFormat](http://msdn.microsoft.com/library/9d9b34e1-03b5-9e56-b9ea-89c7ecce0370%28Office.15%29.aspx)** object for a specified shape range that contains soft edge formatting properties for the shape range. Read-only.


## Version Information

Version Added: Excel 2007 


## Syntax

 _expression_. **SoftEdge**

 _expression_A variable that represents a  **ShapeRange** object.
