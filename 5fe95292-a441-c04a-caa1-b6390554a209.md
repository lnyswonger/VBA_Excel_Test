
# XlOLEType Enumeration (Excel)

Specifies the OLE object type.


## Version Information

Version Added: Excel 2007 



|**Name**|**Value**|**Description**|
|:-----|:-----|:-----|
| **xlOLEControl**|2|ActiveX control|
| **xlOLEEmbed**|1|Embedded OLE object|
| **xlOLELink**|0|Linked OLE object|
