
# AddIns2.Count Property (Excel)

Returns a  **Long** value that represents the number of objects in the collection.


## Version Information

Version Added: Excel 2010 


## Syntax

 _expression_. **Count**

 _expression_A variable that returns a  **AddIns2** object.
