
# ChartTitle.Height Property (Excel)

Returns the height, in points, of the object. Read-only.


## Version Information

Version Added: Excel 2010 


## Syntax

 _expression_. **Height**

 _expression_A variable that represents a  **ChartTitle** object.


### Return Value

Double
