
# Workbook.LockServerFile Method (Excel)

Locks the workbook on the server to prevent modification.


## Version Information

Version Added: Excel 2007 


## Syntax

 _expression_. **LockServerFile**

 _expression_A variable that represents a  **Workbook** object.
