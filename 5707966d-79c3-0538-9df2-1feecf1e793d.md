
# AutoFilter.ApplyFilter Method (Excel)

Applies the specified  **Autofilter** object.


## Version Information

Version Added: Excel 2007 


## Syntax

 _expression_. **ApplyFilter**

 _expression_A variable that represents an  **AutoFilter** object.
