
# SparkPoints.Negative Property (Excel)

Returns a  ** [SparkColor](3de82c5c-eb0a-ab39-64a8-00f4c005c6af.md)** object that represents the color and visibility of markers for negative points of data on a sparkline. Read-only


## Version Information

Version Added: Excel 2010 


## Syntax

 _expression_. **Negative**

 _expression_A variable that represents a  ** [SparkPoints](382bf292-7824-179f-e254-1b72dfb557b2.md)** object.


### Return Value

 **SparkColor**


## Remarks

 The **Negative** property corresponds to the setting of the **Negative Point** check box in the **Show** section, and of the **Negative Points** item on the **Marker Color** drop-down list in the **Style** section of the ** Sparkline Tools Design** tab of the ribbon.
