
# ShadowFormat.Blur Property (Excel)

Returns or sets the degree of blurriness of the specified shadow. Read/write  **Single**.


## Version Information

Version Added: Excel 2007 


## Syntax

 _expression_. **Blur**

 _expression_A variable that represents a  **ShadowFormat** object.
