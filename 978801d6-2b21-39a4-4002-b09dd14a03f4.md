
# VPageBreak.Type Property (Excel)

Returns or sets a  ** [XlPageBreak](8e8f88fd-d12d-077d-bf36-a9084771fa41.md)** value that represents the page break type.


## Syntax

 _expression_. **Type**

 _expression_A variable that represents a  **VPageBreak** object.
