
# ChartObjects.Cut Method (Excel)

Cuts the object to the Clipboard.


## Syntax

 _expression_. **Cut**

 _expression_A variable that represents a  **ChartObjects** object.


### Return Value

Variant


## Remarks

Only embedded charts can be cut.
