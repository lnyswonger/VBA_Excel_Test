
# XlLayoutRowType Enumeration (Excel)

Specifies the type of layout row.


## Version Information

Version Added: Excel 2007 



|**Name**|**Value**|**Description**|
|:-----|:-----|:-----|
| **xlCompactRow**|0|Compact Row|
| **xlOutlineRow**|2|Outline Row|
| **xlTabularRow**|1|Tabular Row|
