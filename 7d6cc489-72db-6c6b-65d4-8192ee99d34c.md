
# ChartObject.Cut Method (Excel)

Cuts the object to the Clipboard.


## Syntax

 _expression_. **Cut**

 _expression_A variable that represents a  **ChartObject** object.


### Return Value

Variant


## Remarks

Only embedded charts can be cut.
