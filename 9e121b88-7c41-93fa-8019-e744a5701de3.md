
# PivotFilter.Parent Property (Excel)

Returns the parent object for the specified  **PivotFilter** object. Read-only.


## Version Information

Version Added: Excel 2007 


## Syntax

 _expression_. **Parent**

 _expression_A variable that represents a  **PivotFilter** object.
