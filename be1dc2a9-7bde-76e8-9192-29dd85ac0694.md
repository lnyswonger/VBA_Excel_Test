
# ModelChanges.TablesAdded Property (Excel)

Returns a  [ModelTableNames Object (Excel)](70fa4b5b-ebc6-9ac9-de6c-40835b1ea12c.md) collection of table names as strings representing all tables which were added to the model as part of a model operation. Read-only.


## Version information

Version Added: Excel 2013 


## Syntax

 _expression_. **TablesAdded**

 _expression_A variable that represents a  **ModelChanges** object.


## Property value

 **MODELTABLENAMES**
