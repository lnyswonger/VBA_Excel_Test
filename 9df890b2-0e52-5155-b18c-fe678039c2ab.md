
# ListObject.AutoFilter Property (Excel)

Filters a  **table** using the AutoFilter feature. Read-only.


## Version Information

Version Added: Excel 2007 


## Syntax

 _expression_. **AutoFilter**

 _expression_A variable that represents a  **ListObject** object.
