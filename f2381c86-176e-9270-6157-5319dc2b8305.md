
# PivotLine.Position Property (Excel)

Returns or sets the position of the  **PivotLine** object. Read-only.


## Version Information

Version Added: Excel 2007 


## Syntax

 _expression_. **Position**

 _expression_A variable that represents a  **PivotLine** object.
