
# Scenario.Show Method (Excel)

Shows the scenario by inserting its values on the worksheet. The affected cells are the changing cells of the scenario.


## Syntax

 _expression_. **Show**

 _expression_A variable that represents a  **Scenario** object.


### Return Value

Variant
