
# XlEditionType Enumeration (Excel)

Specifies the type of edition to be changed.


## Version Information

Version Added: Excel 2007 



|**Name**|**Value**|**Description**|
|:-----|:-----|:-----|
| **xlPublisher**|1|Publisher|
| **xlSubscriber**|2|Subscriber|
