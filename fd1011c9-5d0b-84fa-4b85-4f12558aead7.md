
# PivotFilters.Parent Property (Excel)

Returns the parent object for the specified  **PivotFilters** object. Read-only.


## Version Information

Version Added: Excel 2007 


## Syntax

 _expression_. **Parent**

 _expression_A variable that represents a  **PivotFilters** object.
