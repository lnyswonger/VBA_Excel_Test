
# SlicerItem.Value Property (Excel)

Returns the value of the slicer item. Read-only.


## Version Information

Version Added: Excel 2010 


## Syntax

 _expression_. **Value**

 _expression_A variable that represents a  ** [SlicerItem](cb93cd82-fc3a-f6b7-ae64-db6312db649d.md)** object.


### Return Value

String
