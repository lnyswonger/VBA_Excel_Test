
# ThreeDFormat.Z Property (Excel)

Returns the Z order of the specified  **ThreeDFormat** object. Read/write **Single**.


## Version Information

Version Added: Excel 2007 


## Syntax

 _expression_. **Z**

 _expression_A variable that represents a  **ThreeDFormat** object.
