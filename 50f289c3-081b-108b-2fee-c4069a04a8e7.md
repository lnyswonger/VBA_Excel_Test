
# RTD Object (Excel)

Represents a real-time data object.


## Remarks

Use the  ** [RTD](e181eb35-d8aa-4f46-3d50-6aa51776be7e.md)** property of the ** [Application](19b73597-5cf9-4f56-8227-b5211f657f6f.md)** object to return a **RTD** object.
