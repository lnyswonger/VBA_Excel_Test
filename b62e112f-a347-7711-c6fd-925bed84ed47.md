
# XlPageOrientation Enumeration (Excel)

Specifies the page orientation when the worksheet is printed.


## Version Information

Version Added: Excel 2007 



|**Name**|**Value**|**Description**|
|:-----|:-----|:-----|
| **xlLandscape**|2|Landscape mode.|
| **xlPortrait**|1|Portrait mode.|
