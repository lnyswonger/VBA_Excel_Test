
# Page.LeftFooter Property (Excel)

Specifies a picture or text to be left aligned in the page footer.


## Version Information

Version Added: Excel 2007 


## Syntax

 _expression_. **LeftFooter**

 _expression_A variable that represents a  **Page** object.
