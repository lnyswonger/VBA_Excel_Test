
# ChartCategory.Name Property (Excel)

Returns a  **String** value that represents the name of the object.


## Version information

Version Added: Excel 2013 


## Syntax

 _expression_. **Name**

 _expression_A variable that represents a  **ChartCategory** object.


## Property value

 **STRING**
