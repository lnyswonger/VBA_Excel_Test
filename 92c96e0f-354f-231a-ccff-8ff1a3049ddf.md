
# ListObject.Sort Property (Excel)

Gets or sets the sort column or columns, and sort order for the  **ListObject** collection.


## Version Information

Version Added: Excel 2007 


## Syntax

 _expression_. **Sort**

 _expression_A variable that represents a  **ListObject** object.
