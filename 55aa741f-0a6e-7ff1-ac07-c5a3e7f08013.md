
# DataLabels.Format Property (Excel)

Returns the  ** [ChartFormat](edac71b7-ed38-6658-2cbf-6493dc1ad3ed.md)** object. Read-only.


## Version Information

Version Added: Excel 2007 


## Syntax

 _expression_. **Format**

 _expression_A variable that represents a  **DataLabels** object.
