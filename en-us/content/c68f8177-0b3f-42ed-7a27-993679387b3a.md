
# PivotLineCells.Count Property (Excel)

Returns the number of items in the  **PivotLineCells** collection. Read-only.


## Version Information

Version Added: Excel 2007 


## Syntax

 _expression_. **Count**

 _expression_A variable that represents a  **PivotLineCells** object.
