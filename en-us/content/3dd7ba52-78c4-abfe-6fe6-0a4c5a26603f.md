
# Window.Type Property (Excel)

Returns or sets a  ** [XlWindowType](eca25770-feb6-3723-9bbd-4d2186b8eafa.md)** value that represents the window type.


## Syntax

 _expression_. **Type**

 _expression_A variable that represents a  **Window** object.
