
# DisplayFormat.Orientation Property (Excel)

Returns a value that represents the text orientation of the associated  ** [Range](b8207778-0dcc-4570-1234-f130532cc8cd.md)** object as it is displayed in the current user interface. Read-only.


## Version Information

Version Added: Excel 2010 


## Syntax

 _expression_. **Orientation**

 _expression_A variable that represents a  ** [DisplayFormat](c70b5d7f-adf1-e539-a32d-12c920af7c7e.md)** object.


### Return Value

Variant
