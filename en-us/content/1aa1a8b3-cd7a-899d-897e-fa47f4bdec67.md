
# AutoFilter.Sort Property (Excel)

Gets the sort column or columns, and sort order for the  **AutoFilter** collection.


## Version Information

Version Added: Excel 2007 


## Syntax

 _expression_. **Sort**

 _expression_A variable that represents an  **AutoFilter** object.
