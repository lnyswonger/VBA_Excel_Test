
# Action.Coordinate Property (Excel)

Returns the coordinate property of the  **Action** object. Read-only


## Version Information

Version Added: Excel 2007 


## Syntax

 _expression_. **Coordinate**

 _expression_A variable that represents an  **Action** object.
