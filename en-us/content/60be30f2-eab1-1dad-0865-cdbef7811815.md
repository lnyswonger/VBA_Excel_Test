
# XlThemeFont Enumeration (Excel)

Specifies the theme font to use.


## Version Information

Version Added: Excel 2007 



|**Name**|**Value**|**Description**|
|:-----|:-----|:-----|
| **xlThemeFontMajor**|2|Major.|
| **xlThemeFontMinor**|1|Minor.|
| **xlThemeFontNone**|0|Do not use any theme font.|
