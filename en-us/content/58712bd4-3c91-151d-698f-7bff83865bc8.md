
# SparklineGroup.DisplayBlanksAs Property (Excel)

## Version information

Version Added: Excel 2013 


## Syntax

 _expression_. **DisplayBlanksAs**

 _expression_A variable that represents a  **SparklineGroup** object.


## Property value

 **XLDISPLAYBLANKSAS**
