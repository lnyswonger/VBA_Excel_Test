
# Chart.BackWall Property (Excel)

Returns a  ** [Walls](9c6f0c5b-dbb8-7d71-44b7-29987e750cd3.md)** object that allows the user to individually format the back wall of a 3-D chart. Read-only.


## Version Information

Version Added: Excel 2007 


## Syntax

 _expression_. **BackWall**

 _expression_A variable that represents a  **Chart** object.
