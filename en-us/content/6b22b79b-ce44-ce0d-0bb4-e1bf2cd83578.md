
# Range.MDX Property (Excel)

Returns the MDX name for the specified  **Range** object. Read-only **String**.


## Version Information

Version Added: Excel 2007 


## Syntax

 _expression_. **MDX**

 _expression_A variable that represents a  **Range** object.
