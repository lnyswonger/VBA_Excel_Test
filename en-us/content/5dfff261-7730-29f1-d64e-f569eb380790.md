
# TableStyle.Delete Method (Excel)

Deletes the  ** [TableStyle](191a5c2c-ecf4-f88a-1639-be7ee9c369c3.md)** object.


## Version Information

Version Added: Excel 2007 


## Syntax

 _expression_. **Delete**

 _expression_A variable that represents a  **TableStyle** object.


### Return Value

Nothing
