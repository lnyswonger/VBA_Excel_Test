
# XlScaleType Enumeration (Excel)

Specifies the scale type of the value axis.


## Version Information

Version Added: Excel 2007 



|**Name**|**Value**|**Description**|
|:-----|:-----|:-----|
| **xlScaleLinear**|-4132|Linear|
| **xlScaleLogarithmic**|-4133|Logarithmic|
