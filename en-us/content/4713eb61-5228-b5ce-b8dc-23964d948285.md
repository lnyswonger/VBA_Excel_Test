
# XlPlatform Enumeration (Excel)

Specifies the platform on which a text file originated.


## Version Information

Version Added: Excel 2007 



|**Name**|**Value**|**Description**|
|:-----|:-----|:-----|
| **xlMacintosh**|1|Macintosh|
| **xlMSDOS**|3|MS-DOS|
| **xlWindows**|2|Microsoft Windows|
