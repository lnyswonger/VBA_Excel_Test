
# ListColumn.Total Property (Excel)

Returns the Total row for a  **ListColumn** object. Read-only.


## Version Information

Version Added: Excel 2007 


## Syntax

 _expression_. **Total**

 _expression_A variable that represents a  **ListColumn** object.
