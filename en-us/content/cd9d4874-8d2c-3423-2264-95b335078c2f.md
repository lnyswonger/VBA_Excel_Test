
# IconSetCondition.Delete Method (Excel)

Deletes the specified conditional formatting rule object.


## Version Information

Version Added: Excel 2007 


## Syntax

 _expression_. **Delete**

 _expression_A variable that represents an  **IconSetCondition** object.
