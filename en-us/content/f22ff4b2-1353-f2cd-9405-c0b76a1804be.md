
# TableObject.Delete Method (Excel)

Deletes the  **TableObject** object.


## Version information

Version Added: Excel 2013 


## Syntax

 _expression_. **Delete**

 _expression_A variable that represents a  [ModelTable Object (Excel)](c853beb6-f2e7-dda0-b33a-8110a6c23de8.md) object.


### Return value

 **VOID**
