
# ColorStop.Color Property (Excel)

Returns or sets the color of the represented object. Read/write


## Version Information

Version Added: Excel 2007 


## Syntax

 _expression_. **Color**

 _expression_A variable that represents a  **ColorStop** object.


### Return Value

Variant
