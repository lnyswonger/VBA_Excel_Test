
# Action.Caption Property (Excel)

Returns the caption assigned to the  **Action** object. Read-only **String**.


## Version Information

Version Added: Excel 2007 


## Syntax

 _expression_. **Caption**

 _expression_A variable that represents an  **Action** object.
