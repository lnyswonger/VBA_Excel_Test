
# ChartCategory.Parent Property (Excel)

Returns the parent object for the specified object. Read-only.


## Version information

Version Added: Excel 2013 


## Syntax

 _expression_. **Parent**

 _expression_A variable that represents a  **ChartCategory** object.


## Property value

 **OBJECT**
