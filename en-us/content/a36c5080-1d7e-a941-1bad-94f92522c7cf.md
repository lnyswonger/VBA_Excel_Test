
# Application.OperatingSystem Property (Excel)

Returns the name and version number of the current operating system â€” for example, "Windows (32-bit) 4.00" or "Macintosh 7.00". Read-only  **String**.


## Syntax

This example displays the name of the operating system.


```

```
