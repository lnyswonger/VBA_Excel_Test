
# ColorStop.Delete Method (Excel)

Deletes the represented object.


## Version Information

Version Added: Excel 2007 


## Syntax

 _expression_. **Delete**

 _expression_A variable that represents a  **ColorStop** object.


### Return Value

Nothing
