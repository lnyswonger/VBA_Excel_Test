
# Workbook.Model Property (Excel)

Returns the top level  **Model** object which is the one Data Model for the workbook. Read-only


## Version information

Version Added: Excel 2013 


## Syntax

 _expression_. **Model**

 _expression_A variable that represents a  **Workbook** object.


## Property value

 **MODEL**
