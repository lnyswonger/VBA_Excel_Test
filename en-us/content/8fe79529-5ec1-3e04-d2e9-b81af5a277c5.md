
# XlTabPosition Enumeration (Excel)

Specifies the first or last tab position.


## Version Information

Version Added: Excel 2007 



|**Name**|**Value**|**Description**|
|:-----|:-----|:-----|
| **xlTabPositionFirst**|0|First tab position.|
| **xlTabPositionLast**|1|Last tab position.|
