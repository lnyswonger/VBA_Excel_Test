
# ShadowFormat.Size Property (Excel)

Returns or sets the size of the specified shadow. Read/write  **Single**.


## Version Information

Version Added: Excel 2007 


## Syntax

 _expression_. **Size**

 _expression_A variable that represents a  **ShadowFormat** object.
