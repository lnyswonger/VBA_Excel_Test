
# AddIn.IsOpen Property (Excel)

Returns  **True** if the add-in is currently open. **Boolean** Read-only


## Version Information

Version Added: Excel 2010 


## Syntax

 _expression_. **IsOpen**

 _expression_A variable that returns an  **AddIn** object.
