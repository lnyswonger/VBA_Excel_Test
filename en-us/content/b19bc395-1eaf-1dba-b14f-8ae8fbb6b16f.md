
# WorksheetFunction.Pi Method (Excel)

Returns the number 3.14159265358979, the mathematical constant pi, accurate to 15 digits.


## Syntax

 _expression_. **Pi**

 _expression_A variable that represents a  **WorksheetFunction** object.


### Return Value

Double
