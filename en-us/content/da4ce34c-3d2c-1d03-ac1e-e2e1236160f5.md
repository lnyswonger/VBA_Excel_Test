
# XlGenerateTableRefs Enumeration (Excel)

Specifies the type of table references.


## Version Information

Version Added: Excel 2007 



|**Name**|**Value**|**Description**|
|:-----|:-----|:-----|
| **xlA1TableRefs**|0|A1 Table References.|
| **xlTableNames**|1|Table Names.|
